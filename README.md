# 🌌 ParticleVerse — Interactive 3D Particle Typography Experience

> A futuristic real-time particle animation experience built using pure HTML5 Canvas and JavaScript.

## ▶️ How It Works

1. Open the project in your browser  
2. Watch the animated 3D particle sphere initialize  
3. Type any word or sentence into the input field  
4. Particles dynamically rearrange into your text  
5. Move your cursor around the particles to interact with them  
6. Double-click anywhere to reset back to sphere mode  

---

## 🧠 Core Concepts Used

### 🌍 Fibonacci Sphere Distribution
Particles are evenly distributed in 3D space using the golden-angle algorithm for smooth spherical formations.

### ⚡ Physics Simulation
Each particle uses:
- Velocity
- Acceleration
- Friction
- Attraction forces

to create smooth and natural movement.

### 🎯 Text Sampling System
Words are rendered onto an offscreen canvas and converted into target particle positions.

### 🖱️ Interactive Mouse Repulsion
Particles dynamically react to cursor movement using force-based displacement calculations.
